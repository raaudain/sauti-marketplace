# Sauti Marketplace
A full stack application with form validation and authentication capabilities.

## Dependencies

### Front End
- React
- Redux
- Axios
- React Router Dom
- React Scripts
- Redux Logger
- Redux Thunk

### Back End
- Node.js
- CORS
- JSON Web Token
- Knex.js
- Express
- CORS
- SQLite 3
- bcrypt
- Helmet
- Jest
- Supertest
